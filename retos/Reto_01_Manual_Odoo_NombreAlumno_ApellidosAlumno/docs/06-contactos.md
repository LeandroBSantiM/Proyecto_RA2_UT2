# 06 — Contactos

El módulo de **Contactos** es una herramienta esencial dentro de Odoo, es el punto de partida para la gestión de clientes, leads y proveedores.  
La información que se ingresa en Contactos permite enlazar presupuestos, facturas y oportunidades comerciales

## 1. Asociación automática persona/empresa

Odoo organiza automáticamente los contactos para evitar la duplicación de datos

- **Identificación automática:** Cuando un contacto se agrega, Odoo intenta identificar si el registro corresponde a una Compañía o a un Usuario

![Foto](../assets/img/06-contactos/FotoContactos01.png)
- **Organización jerárquica:** Si un contacto es un individuo que trabaja en una empresa, Odoo lo asocia automáticamente a la compañía correspondiente

- **Ficha completa:** La ficha de contacto incluye la información, y puede ser editada o completada posteriormente

![Foto](../assets/img/06-contactos/FotoContactos02.png)

## 2. Etiquetas para segmentar

Las etiquetas son una buena práctica para gestionar grandes volúmenes de contactos.

- **Creación y personalización:** Se pueden crear directamente desde la ficha de contacto (por ejemplo, "Agencia" o "Proveedor") y asignarles un color distintivo
- **Segmentación:** Las etiquetas permiten filtrar y buscar contactos rápidamente utilizando el buscador de Odoo

![Foto](../assets/img/06-contactos/FotoContactos03.png)
## 3. Importación CSV

- **Proceso de importación:** En el módulo de Contactos, selecciona la opción **"Importar Registros"**
