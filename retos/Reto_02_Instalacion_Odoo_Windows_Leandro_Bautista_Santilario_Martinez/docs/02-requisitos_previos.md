# 02 — Requisitos previos

Antes de comenzar con la instalación de Odoo en Windows, asegúrate de cumplir con los siguientes requisitos:

- **Sistema operativo:** Windows 10 u 11 actualizado  
- **Permisos:** Acceso como usuario con privilegios de administrador
- **Conectividad:** Conexión estable a Internet para descargar lo necesario
- **Espacio en disco:** Mínimo **5 GB** libres (recomendado 10 GB) 
- **Herramientas opcionales:**  
  - Navegador actualizado
  - Cliente **Git**

**Resultado esperado:** Se confirma la versión del sistema operativo, los permisos de administrador y que el entorno está preparado para iniciar la instalación de Odoo
